POST /api/tasks/{id}/assign/ Assign task to user Project Manager
POST /api/tasks/{id}/add_comment/ Add comment to task Project Member
POST /api/tasks/{id}/change_status/ Change task status Assigned User
Query Parameters for Tasks List:
  • project - Filter by project ID
  • assigned_to - Filter by user ID
  • status - Filter by status (todo, in_progress, review, done, blocked)
  • priority - Filter by priority (low, medium, high, urgent)
  • due_soon - Show tasks due in next 7 days
  • search - Search in title and description
